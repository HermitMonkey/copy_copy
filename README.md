# ⌘ copy_copy (Phoenix)

A privacy-first, local-first macOS clipboard manager equipped with on-device AI summarization and End-to-End Encrypted (E2EE) cloud sync.

## Features
* **Zero-Latency Ingestion:** Captures clipboard history instantly in the background.
* **On-Device NLP Engine:** Pure-Dart extractive summarization generates instant TL;DRs for long articles without API calls.
* **Smart Enrichment:** Automatically extracts page titles, hero images, and parses contextual image galleries from URLs.
* **Smart Attachments:** Intercepts PDF URLs and generates clickable Material Document Cards.
* **Privacy-First Vault:** Powered by a local Isar database. Features a "Nuclear Reset" for instant data wiping.
* **Data Sovereignty:** 1-click JSON export of your entire history to your local filesystem.
* **E2EE Cloud Sync:** AES-GCM encrypted payloads synced to Firebase Firestore for cross-device availability.
* **Mac Polish:** Global Hotkeys (`CMD+Shift+V`), Native System Tray integration, and Launch-at-Login support.
